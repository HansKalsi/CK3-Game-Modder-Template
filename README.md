# CK3-Game-Modder-Template

Objectives:
- Create Queen of Ireland Character, and a random other orthodox greek character
- Make the kingdom of Ireland a single united kingdom in 800 AD
- Change the religion of Ireland to Orthodox
- Make the Kings son's age 28 (year only)
- Create some more duchies and counties of ireland (via irish_titles) 

Bonus Objectives:
- Create a unique historical character for the time periood (1000 AD - 1060 AD), providing research evidence (screenshots and texts) 
- Create a unique religion, customised creatively (via irish_religions)
- Create characters for your custom duchies and counties (via irish_characters)
